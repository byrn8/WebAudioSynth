Oscillator
The Oscillartor is at the core of all syntehsizers and is the component that is responsible for generating the sound in the synthesizer. It does this by generating a sine wave by plotting points on a graph based on what oscillator type has been chosen. The sine wave is the representation of the sound that is produced and depending on the type of oscillator that is being used the sound will be different. Oscillator Type
As mentioned in the introduction paragraph, the type of oscillator will affect the overall sound that is produced. There are 4 main oscillator types which include: 'sawtooth', 'triangle', 'square', and 'sine'. Each of these types will affect the sound in a different way. For example, the 'sawtooth' will create a more rough sound whereas the 'sine' will create a more smooth one. Each type has a diagramatical representation if you want to research into them further but playing around with them should be experience enough to get the hang of the different types.
